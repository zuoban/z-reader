# Security Policy

感谢你帮助改进 Z Reader 的安全性。

## 报告方式

如果你发现了可能影响用户数据、认证、上传处理、依赖供应链或部署安全的问题：

- 不要直接公开提交 issue、PR 或讨论帖
- 优先使用 GitHub 的私密安全报告功能
- 如果仓库暂未开启私密报告，请通过维护者的 GitHub 主页联系方式私下告知

请尽量附上这些信息：

- 影响范围
- 复现步骤
- 可能的攻击路径
- 临时缓解建议

## 响应原则

- 我们会尽量先确认问题，再安排修复
- 在补丁发布前，细节可能不会公开披露
- 修复完成后，会根据影响范围决定是否补充公开说明

## 不属于安全问题的内容

以下问题通常更适合走普通 issue：

- 一般性构建失败
- UI 视觉问题
- 非敏感的功能性 bug
- 第三方服务额度、计费或账号本身的问题
