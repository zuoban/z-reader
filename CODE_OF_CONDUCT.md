# Code of Conduct

为了让 Z Reader 的协作环境保持开放、友好和高效，所有参与者都应遵守以下原则。

## 我们鼓励的行为

- 尊重不同背景、经验和观点
- 聚焦问题本身，给出具体、可执行的反馈
- 在讨论中保持善意，默认对方是在尝试解决问题
- 发现误解时及时澄清，而不是升级冲突
- 对新贡献者保持耐心，帮助别人顺利参与

## 我们不接受的行为

- 人身攻击、羞辱、嘲讽或挑衅
- 歧视、骚扰、威胁或持续性的敌对行为
- 故意泄露他人隐私、凭据或敏感信息
- 与项目协作无关的刷屏、引战或恶意行为

## 适用范围

本准则适用于仓库内的 issue、PR、讨论、代码评审，以及与项目相关的其他公共协作场景。

## 维护者处理方式

维护者可以对不当行为做出提醒、删改内容、关闭讨论或限制参与等处理。
如果你认为有行为违反本准则，请私下联系维护者。
